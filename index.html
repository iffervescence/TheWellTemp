The only thing needed to enable TheWell are small boxes not much larger than a 330ml drink can, and at around the same weight too.

Description of main features

Security

TheWell boxes are secure from external interference. We do this by authenticating each box with our own cloud servers. This means that you can be assured that the content being distributed is free from harmful or undesirable media.

Broadcast media

TheWell boxes work as hotspots that users can connect to, for up to 15 users simultaneouly. Media can be easily broadcasted directly to the user's devices, without a need for physical cables for transfer or cumbersome downloading processes.

Data management

TheWell can be used to teach courses - distributing course materials, and collecting completed assignments (?). And on top of that, it allows management of these courses - giving data analytics of how the courses are progressing, as well as how the students are faring. There is much more data analytics that can be done, and you can find out more about it here.
