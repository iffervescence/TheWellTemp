More info on main features
Security
TheWell boxes are secure from external interference. We do this by authenticating each box with our own cloud servers. This means that you can be assured that the content being distributed is free from harmful or undesirable media.
TheWell boxes work as hotspots that users can connect to, for up to 15 users simultaneouly. Media can be easily broadcasted directly to the user's devices, without a need for physical cables for transfer or cumbersome downloading processes.
